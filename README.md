# Epic Games

**Epic Games** is a web application designed for downloading exciting games, exploring new games, and staying updated with interesting gaming news. The platform provides a user-friendly interface to browse games, customize profiles, and manage account settings effortlessly.

## Overview

Epic Games allows users to register on the website and access a wide library of games. Users can customize their profiles with a preferred name and image, explore new game releases, and even reset their password if forgotten. The application is fully responsive and interactive, offering a smooth and engaging user experience.

## Key Features

- **User Registration & Authentication:** Create an account to access all features.  
- **Game Library Exploration:** Browse and discover exciting games.  
- **Profile Customization:** Set your profile name and image.  
- **Password Reset:** Easily reset your password via your registered email.  
- **Interactive UI:** Smooth animations and transitions for enhanced experience.

## Technology Stack

- **Frontend:** React, Vite  
- **Styling & UI:** Tailwind CSS, Framer Motion, Swiper  
- **Backend & Authentication:** Firebase  
- **Routing & State Management:** React Router, React Toastify  
- **Icons:** React Icons
